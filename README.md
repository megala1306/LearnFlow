# 🧬 LearnFlow: Premium Adaptive Learning

> **Adaptive Intelligence for Accelerated Learning & Permanent Retention.**

LearnFlow is a state-of-the-art multimodal learning platform designed to optimize the human learning process. By integrating **Reinforcement Learning (RL)** with the **Ebbinghaus Forgetting Curve**, LearnFlow dynamically adjusts to your unique cognitive footprint, ensuring you learn faster and remember longer.

---

## ✨ Core Intelligence Layers

### 🧠 Adaptive Complexity Engine
The system doesn't just present content; it evaluates your performance in real-time. Content blocks (Easy, Medium, Hard) are dynamically unlocked or adjusted based on your mastery of fundamental concepts.

### 📈 Predictive Retention (RL Engine)
Leveraging a sophisticated **Q-Learning Agent**, LearnFlow decides the optimal moment for review:
- **Reward Scaling**: Higher accuracy and faster response times yield exponential XP rewards.
- **Forgetting Curve Integration**: $R = e^{-kt}$ modeling estimates your current retention level.
- **Strategic Intervention**: The system suggests `No Review`, `Light Refresh`, or `Immediate Reinforcement`.

### ⚡ VARK Multimodal Architecture
Content is automatically tailored to your preferred learning modality:
- **Visual**: Infographic and video-based modules.
- **Aural**: Auditory explanations and mnemonic cues.
- **Read/Write**: Technical documentation and synthetic summaries.
- **Kinesthetic**: Interactive "Action Nodes" and assessments.

## 🏗️ Technical Architecture

- **Frontend**: React (Vite) + Framer Motion (Ceramic Interactions) + Tailwind CSS (Neural Grid).
- **Backend API**: Node.js & Express (Scalable Logic Layer).
- **Intelligence Core**: Python (FastAPI) + NumPy (RL Calculations).
- **Database**: MongoDB (Flexible JSON Storage).

---

## 🛠️ Deployment Protocol

### ⚡ Automatic Execution
Run the following streams to initialize the mainframe:

1. **Backend Layer**:
   ```bash
   cd backend && npm install && node seed.js && npm start
   ```

2. **Intelligence Layer (ML)**:
   ```bash
   cd ml-service && pip install -r requirements.txt && python main.py
   ```

3. **Interface Layer (Frontend)**:
   ```bash
   cd frontend && npm install && npm run dev
   ```

## 📊 Cognitive Logic Flow
1. **Interaction**: User engages with a `LearningUnit` assessment.
2. **Analysis**: Performance data is beamed to the Backend.
3. **Optimizing**: The ML Service updates the **Q-Table** and estimates the **Forgetting Vector**.
4. **Adaptive Response**: The mainframe unlocks the next logical progression in the curriculum matrix.
